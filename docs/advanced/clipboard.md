# 剪贴板
小企鹅可以暂存最近复制的若干条文本。某些情况也支持暂存主选区内容。

## 设置

### 触发键
按 `Ctrl+semicolon` 调出剪贴板。

### 粘贴主选区
在 Linux 上，选中的文字会自动暂存在主选区中，无需手动复制即可通过鼠标中键粘贴。

在 macOS 上没有主选区的概念。对于大多数应用，小企鹅仅能以一定时间间隔获取当前输入框内选中的文字，作为主选区内容。

::: info
Terminal 和 iTerm 是已知的可获取任意选中文字的应用。
:::

### 项目个数
最多暂存几条剪贴板记录。

每页显示 5 条，翻页、移动高亮的快捷键在 `全局配置` -> `快捷键` 设置。

### 不要显示密码管理工具中的密码
开启此选项，则当您从密码管理器（如 Bitwarden）中复制敏感信息时，它将不会被小企鹅暂存。

::: info
是否将一段文本视为敏感信息由密码管理器决定。若密码管理器未实现标记敏感信息的特性，则小企鹅会暂存所有复制的文本。
:::

### 以普通文本方式显示密码
* 开启此选项，则密码将以明文形式显示在剪贴板中。
* 关闭此选项，则密码在剪贴板中显示时将以 `••••••••` 代替。

### 自动清除密码的秒数
密码暂存后多久自动清除。设为 0 则不自动清除。
