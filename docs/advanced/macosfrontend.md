# macOS 前端
一些 macOS 平台特色选项。

## 状态栏
小企鹅在状态栏注册了一个文字按钮用于显示当前的输入法。例如英文键盘时显示 `en`，拼音时显示 `拼`。
* 菜单：点击按钮时，展开一个菜单，可以用来切换输入法的激活状态或隐藏输入法；
* 切换输入法：点击按钮时没有菜单，直接切换激活状态；
* 隐藏：不显示按钮，通常用于状态栏空间紧张时。

## 应用默认输入法
每次聚焦应用时使用的输入法，默认为进入 Terminal 时自动切换为英文。

下拉菜单内置了不好寻找的 Spotlight（聚焦搜索）和 CharacterPallette（按 `fn+E` 唤出的 emoji 选择器），旁边的按钮可用于选择其他应用。

::: warning
在您未更改全局配置的情况下，一旦您设置某应用使用英文，则无法用 `Shift_L` 键切换为中文，需要使用 `Ctrl+Shift_L`。

您可在 `全局配置` -> `快捷键` 中删去 `临时切换输入法` 的 `Shift_L` 并在 `切换输入法` 添加 `Shift_L`，以使用 `Shift_L` 切换到中文。
:::

## Vim 模式
此选项仅针对程序员。当您在 Vim 中的插入模式按 `Esc`/`Ctrl+[`/`Ctrl+C` 进入普通模式时，可自动切换为英文输入法。

这是基于按键检测的技术，因此会有错判（您未在 App 使用 Vim，但按下了这三个键之一）和漏判（应用拦截了按键，比如 Terminal 不给输入法发送 `Esc`）情况，请酌情使用。

更精确的基于 Vim 主动向输入法报告状态的技术，请见 [fcitx5-remote](../topic/cli#fcitx5-remote)。

见[并击](../im/rime#并击)。

## 监视剪贴板
macOS 的任何应用都可以获取当前剪贴板中的内容，但只能以一定时间间隔主动获取，而不能在您复制一段文字时自动接收。

剪贴板的使用方式见[剪贴板](./clipboard)。

## 清除追踪参数
当您复制一段 URL（例如 https://example.com/search?page=1&sid=xyz ）时，可能有一些参数有实际意义（`page=1`），而另一些参数仅用来标记您（此处为 `sid=xyz`）。

开启此选项可以在小企鹅的剪贴板中去除一些已知的追踪参数（不改变系统剪贴板中的内容）。
