# 标点
按下标点键输出全角（中文）标点。若要配置中英标点的对应关系，见[拼音/双拼](../im/pinyin#标点)。

## 设置

### 切换键
按下 `Ctrl+period` 切换全角/半角标点。

### 字母或者数字之后输入半角标点
在全角标点下，
* 开启此选项，则数字或字母后紧跟的逗号、句号自动使用半角；此时按 `Backspace` 将删除这个半角标点，并自动替换为全角标点（Terminal 等应用不支持替换）；
* 关闭此选项，则一律使用全角标点。

### 同时输入成对标点（例如引号）
在全角标点下，
* 开启此选项，则按引号键同时上屏`‘’`（系统限制无法做到光标居中）；
* 关闭此选项，则按引号键上屏 `‘`，再按则上屏 `’`。
